X-Axis2

X2 Axis parameters

setMaxX2 float
Maximum value to set the data in the 2nd axis
Default: null
Used in: AreaLine, BarLine, StackedLine, StackedPercentLine
setMinX2 float
Minimum value to set the data in the 2nd axis
Default: null
Used in: AreaLine, BarLine, StackedLine, StackedPercentLine
xAxis2AbsMax float Private
Minimum non-filtered value for the data plotted in the 2nd X axis
Default:
Used in: Area, AreaLine, Bar, BarLine, Boxplot, Candlestick, Correletion, DotLine, Dotplot, Heatmap, Line, Pie, Stacked, StackedLine, StackedPercent, StackedPercentLine
xAxis2Exact boolean
Flag to force exact values for the data in the 2nd X axis
Default: false
Used in: AreaLine, BarLine, StackedLine, StackedPercentLine
xAxis2MaxStrLength string Private
Longest string of all the values in the ticks in the 2nd X axis
Default:
Used in: Area, AreaLine, Bar, BarLine, Boxplot, Candlestick, Correletion, DotLine, Dotplot, Heatmap, Line, Pie, Stacked, StackedLine, StackedPercent, StackedPercentLine
xAxis2MinorValues array
Values for the minor ticks in the 2nd X axis
Default: []
Used in: AreaLine, BarLine, StackedLine, StackedPercentLine
xAxis2Show boolean
Flag to show on not the values of the ticks in the second X axis
Default: true
Used in: Area, AreaLine, Bar, BarLine, Boxplot, Candlestick, Correletion, DotLine, Dotplot, Heatmap, Line, Pie, Stacked, StackedLine, StackedPercent, StackedPercentLine
xAxis2TickFormat string
Format for the tick values in the second X axis
Default: false
Used in: Area, AreaLine, Bar, BarLine, Boxplot, Candlestick, Correletion, DotLine, Dotplot, Heatmap, Line, Pie, Stacked, StackedLine, StackedPercent, StackedPercentLine
xAxis2Ticks integer Private
Number of ticks in the 2nd X axis. It is the same as xAxisTicks.
Default: 5
Used in: Area, AreaLine, Bar, BarLine, Boxplot, Candlestick, Correletion, DotLine, Dotplot, Heatmap, Line, Pie, Stacked, StackedLine, StackedPercent, StackedPercentLine
xAxis2Title string
Title for the 2nd X axis
Default: false
Used in: Area, AreaLine, Bar, BarLine, Boxplot, Candlestick, Correletion, DotLine, Dotplot, Heatmap, Line, Pie, Stacked, StackedLine, StackedPercent, StackedPercentLine
xAxis2Values array
Values for the ticks in the 2nd X axis
Default: []
Used in: AreaLine, BarLine, StackedLine, StackedPercentLine
