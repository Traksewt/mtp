R-Axis

R axis parameters

rAxisAbsMax float Private
Maximum non-filtered value for the data plotted in the R axis
Default:
Used in: Circular
rAxisAbsMin float Private
Minimum non-filtered value for the data plotted in the R axis
Default:
Used in: Circular
rAxisLabelOrientation option
Orientation to show the ticks values in circular plots.
Default: perpendicular
Options: perpendicular, circular
Used in: Circular
rAxisMaxStrLength string Private
Longest string of all the values in the ticks in the R axis
Default:
Used in: Circular
rAxisMinorValues array
Values for the minor ticks in the R axis
Default: []
Used in: Circular
rAxisShow boolean
Flag to show on not the values of the ticks in the R axis
Default: true
Used in: Circular
rAxisTickColor color
Color for the tick lines in the R axis
Default: rgb(0,0,0)
Used in: Circular
rAxisTicks integer
Number of ticks in the R axis
Default: 5
Used in: Circular
rAxisValues array
Values for the ticks in the R axis
Default: []
Used in: Circular
